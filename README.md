
# Luso-Business - Centre de formation certifié Qualiopi

Luso-Business est un centre de formation certifié Qualiopi qui propose des formations professionnelles adaptées à vos besoins. Notre mission est de transformer vos entreprises en les rendant plus compétitives grâce à nos formations de pointe.

## Technologies utilisées

Ce projet utilise React, Vite et Tailwind CSS pour la création d'une application web moderne. Nous avons choisi ces outils en raison de leurs performances rapides au développement (HMR), de leur facilité d'utilisation avec React et de leur puissance dans le domaine du design responsive.

## Installation

1. Clonez le dépôt sur votre machine :
   ```bash
   git clone https://github.com/your-repo/luso-business.git
   cd luso-business
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez l'application en mode développement :
   ```bash
   npm run dev
   ```

4. Pour la production, construisez le projet :
   ```bash
   npm run build
   ```

5. Déployez le contenu de la directory `dist` sur votre serveur.

## Configuration ESLint

Pour une meilleure qualité du code et pour une application en production, nous recommandons l'utilisation de TypeScript avec les règles de linting à type. 

- Pour intégrer TypeScript, utilisez notre template [React + TypeScript](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts).

- Pour configurer ESLint avec `typescript-eslint`, suivez la documentation officielle : 
  - [typescript-eslint](https://typescript-eslint.io)
